  • 에어컨 필터 점검
    • 시동을 끄고 차량 내부 온도가 식은 후에 점검
    • 필터가 매우 오래되거나 오염이 심한 경우 교체
    • 에어컨 성능을 확인하고 에어흐름과 온도가 정상인지 확인
  • 냉각수 점검
    • 엔진을 가동시켜 냉각수 온도가 정상 범위 내에 있는지 확인
    • 정차 시 바닥에 냉각수 누수 흔적이 있는지 확인
    • 오래됐거나 냉각수에 오염물이 많이 섞인 경우 냉각수를 교체
